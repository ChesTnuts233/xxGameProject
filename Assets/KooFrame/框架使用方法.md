# 框架使用方法 


框架的更新信息会显示在框架的父文件夹的Inspector窗口里

游戏开始/游戏门户
GameRoot 在框架的Prefab中   
负责管理背包系统 网格系统 放置系统

单例：

> * 普通脚本继承KooSingleton  
> * Mono脚本继承KooSingletonMono

事件中心：这个是解耦用的不多说

````csharp
事件的注册：
//这里info是方法的参数。。现在还只能传一个参 后面会完善多参。。。。
EventCenter.Instance.AddEventListener<T>("事件名",方法名/*（info）*/);   
 
事件的触发
EventCenter.Instance.EventTrigger<T>("事件名",info);  
````

对象池:
要被对象池回收的对象需要继承PoolObject
````csharp
对象池初始化：
//对象池里拿出东西       //Resource里的路径
PoolMgr.Instance.GetObj("xxxx", (go) =>
{
	//对go的操作
});

//对象池放回东西             //对象名                 //对象
PoolMgr.Instance.PushObj(/*this.gameObject.name*/, /*this.gameObject*/);
````

资源管理
````csharp                          
//异步加载 同步同理LoadAsync改为Load即可   //路径名称
ResMgr.Instance.LoadAsync<GameObject>(pathName, (go) =>
{
     //对go的操作
});
````

UI系统
````csharp
UIPanel 统一继承KooBasePanel 放置在Resource的UI路径下
打开Panel             //PanelType在UIPanelType枚举中声明  //层级
 UIMgr.Instance.OpenPanel<PanelType>("LoginPanel", UIMgr.E_UI_Layer.Bottom, (panel) => { 
       //对Panel的操作
 });
关闭Panel
UIMgr.Instance.ClosePanel("PanelName");
````
